# morphogenesis

> **Form from formlessness. Turing patterns for agent development.**

[![crates.io](https://img.shields.io/crates/v/morphogenesis.svg)](https://crates.io/crates/morphogenesis)
[![license](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Reaction-diffusion systems for agent morphogenesis — how agents develop structure from initially homogeneous states. Implements Turing patterns, activator-inhibitor systems, and Belousov-Zhabotinsky dynamics.

## Turing Patterns

Alan Turing's 1952 paper "The Chemical Basis of Morphogenesis" showed how two interacting chemicals (activator and inhibitor) diffusing at different rates can spontaneously create patterns. This library applies the same principle to agent development:

- How do agents specialize from a uniform pool?
- How do new behavioral patterns emerge?
- How do boundaries form between agent roles?
